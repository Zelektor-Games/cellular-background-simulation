# Cellular Background Simulation

Hi, this is Alex Lowen. I couldn't find any decent free footage of cells under a microscope for my video, so I spent 3 hours in Unity in the kitchen creating a little “simulation” myself. It's not a full-fledged game, but it's a decent foundation for one, and it'll work for the video.

## Features:
* Randomly generated cells
* Animations and particles
* Glow and post-processing effects for a stylish "microscope" look.
* Automated cell life cycle (birth, movement, combat, and death)
* If there are too many cells, some of them get sick, and half of them die.

## How to use
You can open this project in Unity (Version 6000.3.10f1), tweak the particle systems, or change the spawn rates in the inspector to customize your own footage or game from it.

It’s a highly satisfying sight with simple rules for cell life.
You can do whatever you want with this project; I decided to post it, but just know that this is Alex Lowen’s first open-source project ;)
